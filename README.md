# IPBAN

- Limit server output to IP countries
- Support TCP,UDP
- Support IPv4, IPv6
- Support Multi Port
- Support Multi Country
- Persistent settings after reboot
- Automatic IP update every 2 days
- Tested on Ubuntu version 22
